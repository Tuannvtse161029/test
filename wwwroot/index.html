Mastering Scopus Search Query Language

Elsevier Scopus uses a rich boolean Search query syntax. By using specific field codes, you can filter documents with extreme precision. Here is a curated reference sheet of standard field codes and boolean combinations.

Essential Scopus Field Codes

Field Code Purpose Example Search Query
TITLE-ABS-KEY(...) Search within Document Title, Abstract, or Keywords (Most Common). TITLE-ABS-KEY("deep learning")
AUTHOR-NAME(...) Search for works by author last name and first initials. AUTHOR-NAME(hawking, s)
AFFIL(...) Search for works affiliated with a specific university or institute. AFFIL(University of Oxford)
KEY(...) Match specific indexing keywords defined by researchers. KEY("graphene oxide")
PUBYEAR IS ... Restrict papers to a specific year. (Or use operators like >, <). TITLE-ABS-KEY(quantum) AND PUBYEAR IS 2023
DOI(...) Search specifically for an article by its digital object identifier. DOI("10.1038/nature14539")

Boolean Operators & Examples

AND (Intersection)

Filters articles containing both search terms.

TITLE-ABS-KEY("cancer") AND TITLE-ABS-KEY("immunotherapy")
OR (Union)

Filters articles containing at least one of the terms.

KEY("deep learning") OR KEY("neural networks")
AND NOT (Exclusion)

Excludes articles containing the second term.

TITLE-ABS-KEY("renewable energy") AND NOT KEY("solar")
